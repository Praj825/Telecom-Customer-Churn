# 📡 Telecom Customer Churn Dashboard | Power BI

## 📌 Project Overview
Analyzed **7,043 telecom customer records** to identify churn patterns and retention opportunities.

## 🛠️ Tools Used
- **Power BI** (DAX, Power Query)
- **Dataset:** Telco Customer Churn Dataset (Kaggle)

## 📊 Key Insights
| # | Insight |
|---|---|
| 1 | Overall Churn Rate: **26.5%** (1,869 churned customers) |
| 2 | **Month-to-month contracts** — highest churn segment (3.9K) |
| 3 | **Fiber optic users** show highest churn rate |
| 4 | **Electronic check** payment method linked to highest churn |
| 5 | Low tenure customers **(0–10 months)** most at risk |

## 📈 Dashboard Features
- KPI cards — Churn Rate, Total Customers, Retained vs Churned
- Contract type churn breakdown
- Internet service analysis
- Payment method distribution
- Tenure vs churn trend
- Interactive slicers for Contract, Internet Service, Gender

## 💡 Business Recommendations
- Offer discounts to convert month-to-month → long term contracts
- Investigate fiber optic service quality issues
- Target new customers (0–10 months) with loyalty programs
